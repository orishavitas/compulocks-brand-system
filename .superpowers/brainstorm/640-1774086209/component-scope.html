What should the plugin generate on the Figma canvas?

From your git tokens → what gets manufactured in Figma

A

Style Guide Page

Color swatches, typography specimens, spacing scale — a visual token reference. No interactive components, just the raw design language documented visually.

B

UI Components

Actual Figma components — buttons, cards, inputs, badges — built from the tokens. Designers can drag these into their designs directly.

C

Both — Style Guide + Components

Two pages: one token reference page, one component library. Both auto-generated and kept in sync with git. The complete design system.
